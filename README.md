# 🎮 Terminal Hacker Game

## 📌 About This Project

這是我用 Python 和 Pygame 做的一個小遊戲。
整體風格是「終端機 + 可愛角色」，我想做出一種看起來很科技、但玩起來不會太嚴肅的感覺。

---

## 🎯 Why I Made This

一開始是因為對遊戲開發有興趣，所以想自己做做看。
我先跟著 YouTube 教學做出基本版本，熟悉整個架構之後，再慢慢加入自己的想法，改成現在這個樣子。

### 參考連結
["pygame 3小時製作一個遊戲 "](https://youtu.be/61eX0bFAsYs?si=dyNC_Gf1r69u_Ibu)

## 🛠️ What I Did

我沒有只是照著做，而是：

- 修改原本的遊戲邏輯

- 加入自己的遊戲機制

- 調整整體風格（變成終端機 + 可愛角色）

- 遊戲裡的圖片都是我自己畫的

## 🎮 How to Play

### ✅ 最簡單的方法（推薦）

1. 下載整個遊戲資料夾並解壓縮

2. 直接點擊 `.exe` 檔就可以玩

⚠️ 注意：

exe 檔需要和其他檔案（圖片、音效）放在同一個資料夾裡

👉 不要單獨把 exe 拉出來，否則遊戲可能會無法正常運作

---

### 🧪 用 Python 執行（如果你有安裝環境）

```bash

pip install -r requirements.txt

python main.py

```
